# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

## [1.0.0] - 2026-01-15

### ✨ Agregado
- **Sitio web completo** para DrHousePcclinic
- **Navbar responsiva** con menú mobile
- **Hero section** con video de fondo y animaciones
- **Marquee ticker** con ofertas animadas
- **6 servicios principales** con cards animadas
- **8 consolas compatibles** con detalles específicos
- **Carrusel de promociones** con 5 banners auto-play
- **Galería interactiva** con 32 fotos/videos y filtros
- **Sección de precios** con "valores a consultar"
- **6 testimonios** de clientes satisfechos
- **8 preguntas frecuentes** con acordeones
- **Formulario de contacto** funcional
- **Footer completo** con links
- **Botón WhatsApp flotante** con animación
- **4 tipos de banners publicitarios** con animaciones
- **Efectos de fondo tecnológico** (grid, circuitos, scanlines)
- **Deploy automático** a GitHub Pages

### 🎨 Diseño
- Tema oscuro profesional con neón púrpura/cyan
- 100% responsive (móvil, tablet, desktop)
- Animaciones fluidas con Framer Motion
- Gradientes y efectos de glow
- Iconos de Lucide React

### 🛠️ Tecnología
- React 19 con TypeScript
- Vite como build tool
- Tailwind CSS 4
- Framer Motion para animaciones
- GitHub Actions para CI/CD

### 📱 Características
- Integración WhatsApp en múltiples puntos
- Galerías con lightbox
- Carruseles automáticos
- Animaciones de entrada
- Efectos de hover interactivos
- Formas contacto validadas

## Cómo reportar cambios

Cada commit sigue esta estructura:
- `feat:` Nueva funcionalidad
- `fix:` Corrección de bug
- `docs:` Documentación
- `style:` Estilos/formato
- `refactor:` Refactorización
- `test:` Tests
- `chore:` Tareas
