# Contribuir a DrHousePcclinic

¡Gracias por tu interés en contribuir! 🎮

## 🚀 Cómo Contribuir

1. **Fork** el repositorio
2. **Clona** tu fork: `git clone https://github.com/tu-usuario/drhousepcclinic.git`
3. **Crea una rama**: `git checkout -b feature/nueva-funcionalidad`
4. **Haz tus cambios** y commitea: `git commit -m "Agrega nueva funcionalidad"`
5. **Push** a tu fork: `git push origin feature/nueva-funcionalidad`
6. Abre un **Pull Request**

## 📝 Convenciones

### Commits
Usamos commits semánticos:
- `feat:` Nueva funcionalidad
- `fix:` Corrección de bug
- `docs:` Documentación
- `style:` Cambios de estilo (formato, espacios)
- `refactor:` Refactorización de código
- `test:` Tests
- `chore:` Tareas de mantenimiento

### Código
- TypeScript estricto
- Componentes funcionales con hooks
- Tailwind CSS para estilos
- Nombres de archivos en PascalCase para componentes

## 🐛 Reportar Bugs

Abre un Issue con:
1. Descripción del problema
2. Pasos para reproducir
3. Comportamiento esperado vs actual
4. Screenshots si aplica

## 💡 Sugerir Mejoras

Abre un Issue con la etiqueta `enhancement` describiendo tu idea.
