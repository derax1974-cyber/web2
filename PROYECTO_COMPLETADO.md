## 🚀 Cómo Subir a GitHub (Resumen)

### En 5 pasos:

```bash
# 1. Configura Git (primera vez)
git config user.name "Tu Nombre"
git config user.email "tu@email.com"

# 2. Inicializa el proyecto
git init

# 3. Agrega archivos y commit
git add .
git commit -m "feat: sitio web DrHousePcclinic completo"
git branch -M main

# 4. Conecta con GitHub (cambia TU-USUARIO)
git remote add origin https://github.com/TU-USUARIO/drhousepcclinic.git

# 5. Sube a GitHub
git push -u origin main
```

### Luego activa GitHub Pages:
1. Ve a tu repo en GitHub → Settings
2. En el menú, selecciona Pages
3. Source: GitHub Actions
4. ¡Listo! Sitio en: `https://TU-USUARIO.github.io/drhousepcclinic/`
